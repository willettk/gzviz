An Animated Squarified TreeMap for Galaxy Zoo 2

Hover over a box to see an example image of a particular galaxy class.

Refresh the page to bring back the full tree.

The size of each box is proportional to the number of galaxies in it; the color is proportional to the complexity of that class.

The code at the top of each box ('Ei','Sb?t', etc) gives the galaxy morphology class measured from the user consensus.

A full description is in the Appendix of the paper by Willett et al. (2013)

The data for Galaxy Zoo 2 can be downloaded at data.galaxyzoo.org; code for this figure is here.
